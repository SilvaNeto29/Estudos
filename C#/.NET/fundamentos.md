# .NET Framework

- É uma junção de bibliotecas do C# que foram unidas para dentro de um framework.
- **SOMENTE COMPATÍVEL COM WINDOWS**

# .NET Core (Mais Moderno 2015 <)

- Lançado para ter compatibilidade com tudo, inclusive Mac e Linux.
- Core é pq foi feito pra ter somente o essencial.

# .NET Standard

- É o que garante compatibilidade entre o Core, Framework, XAMARIN...

# Hoje já não tem mais separação entre core e framework (v5 <)

# Runtime e SDK

- Runtime é pra rodar a aplicação
- SDK é o kit pra conseguir desenvolver as aplicações em .NET
